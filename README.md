# Pearson Edexcel Pseudocode Interpreter

This is a fun project to bring the Pearson Edexcel pseudocode to life! The interpreter is designed to execute pseudocode written in the style used in Pearson Edexcel IGCSE Computer Science course. It mostly follows the syntax and conventions outlined in the [Edexcel specification](https://qualifications.pearson.com/content/dam/pdf/International%20GCSE/Computer%20Science/2016/Specification%20and%20sample%20assessments/International_GCSE_Computer_Science_Specification.pdf).

## Features

- Supports basic data types: integers, real numbers, strings, and booleans.
- Implements control structures: IF statements, FOR loops, WHILE loops.
- Allows for variable assignment and simple input/output operations.
- Provides error handling for syntax errors and runtime errors.
